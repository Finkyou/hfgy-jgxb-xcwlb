# 项目结构说明

```
das/                        # 网站前端主目录
├── css/                    # 样式表目录
│   ├── core.css            # 全站基础样式，含Bootstrap引入、自定义变量，服务所有页面
│   ├── style.css           # 全站美化与自定义样式，服务所有页面
│   ├── auth.css            # 登录、注册、找回密码等认证相关页面样式
│   ├── bootstrap.min.css   # Bootstrap 4/5 样式库（部分页面备用）
│   ├── core.css            # 全站基础样式（重复，实际只需一个）
│   ├── duty.css            # duty.html 值班表页面样式
│   ├── loading.css         # loading.html 加载动画页面样式
│   ├── members.css         # members.html 成员介绍页面样式
│   ├── news.css            # news.html 新闻页面样式
│   ├── photography.css     # photography.html 摄影展区页面样式
│   ├── recruitment.css     # recruitment.html 招新页面样式
│   ├── resources.css       # resources.html 资源申请页面样式
│   ├── responsive.css      # 全站响应式适配样式，服务所有页面
│   ├── style.backup.css    # 样式备份文件
│   └── submit.css          # submit.html 投稿页面样式
├── data/                   # 静态内容数据（由Python脚本生成，前端自动读取）
│   ├── news.json           # 新闻内容数据，供news.html动态渲染
│   └── gallery.json        # 摄影展区内容数据，供gallery.html动态渲染
├── img/                    # 图片资源目录
│   └── gallery/            # 摄影展区图片存放目录，供gallery.html使用
├── js/                     # 前端JS脚本
│   ├── main.js             # 全站表单校验与交互优化，服务所有页面
│   ├── admin.js            # admin.html 后台管理页交互逻辑
│   ├── profile.js          # profile.html 用户个人主页交互逻辑
│   ├── duty.js             # duty.html 值班表页面交互
│   ├── loading.js          # loading.html 加载动画页面交互
│   ├── news.js             # news.html 新闻页面交互
│   ├── photography.js      # photography.html 摄影展区页面交互
│   ├── recruitment.js      # recruitment.html 招新页面交互
│   ├── resources.js        # resources.html 资源申请页面交互
│   ├── submit.js           # submit.html 投稿页面交互
├── pages/                  # 主要功能页面
│   ├── blog.html           # 博客/公告页面
│   ├── duty.html           # 值班表页面
│   ├── forgot.html         # 找回密码页，用户通过邮箱重置密码
│   ├── login.html          # 登录页，用户输入账号密码登录
│   ├── members.html        # 成员介绍页面
│   ├── news.html           # 新闻动态页，自动渲染news.json，展示所有新闻
│   ├── photography.html    # 摄影展区页，自动渲染gallery.json，展示所有摄影作品
│   ├── privacy.html        # 隐私政策页面
│   ├── profile.html        # 用户个人主页，查看和管理个人投稿、资源、订阅、信息
│   ├── recruitment.html    # 招新报名页面，用户报名参加外联部
│   ├── register.html       # 注册页，新用户注册账号
│   ├── resources.html      # 资源申请页面，用户申请资源
│   ├── submit.html         # 投稿区页面，用户提交稿件
├── index.html              # 网站首页，导航入口
├── generate_news_json.py   # 新闻内容生成脚本（运行后生成data/news.json）
├── generate_gallery_json.py# 摄影展区内容生成脚本（运行后生成data/gallery.json）
└── 项目结构说明.md         # 本文件，项目结构和说明
```

---

- **css/**：存放全站和各页面的样式文件，每个css文件后标注其主要服务页面。
- **data/**：存放新闻、摄影展区等内容的JSON数据，前端页面自动读取渲染。
- **img/gallery/**：摄影展区图片，供photography.html和gallery.html使用。
- **js/**：存放各页面的交互脚本，每个js文件后标注其主要服务页面。
- **pages/**：每个页面单独标注其用途。
- **index.html**：网站首页，导航入口。
- **generate_news_json.py / generate_gallery_json.py**：内容管理脚本，自动生成内容数据文件。
- **项目结构说明.md**：本文件，项目结构和说明。
