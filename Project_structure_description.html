项目结构说明

das/                        # 网站前端主目录
├── css/                    # 样式表目录
│   ├── core.css            # 全站基础样式，含Bootstrap引入，服务所有页面
│   ├── style.css           # 全站美化与自定义样式，服务所有页面
│   ├── auth.css            # 登录、注册、找回密码等认证相关页面样式
│   ├── bootstrap.min.css   # Bootstrap 4/5 样式库（可选）
│   ├── duty.css            # duty.html 值班表页面样式
│   ├── loading.css         # loading.html 加载动画页面样式

│   ├── news.css            # news.html 新闻页面样式
│   ├── photography.css     # photography.html 摄影展区页面样式
│   ├── recruitment.css     # recruitment.html 招新页面样式
│   ├── resources.css       # resources.html 资源申请页面样式
│   ├── responsive.css      # 全站响应式适配样式，服务所有页面
│   ├── style.backup.css    # 样式备份文件
│   └── submit.css          # submit.html 投稿页面样式
├── data/                   # 静态内容数据（由Python脚本生成，前端自动读取）
│   ├── news.json           # 新闻内容数据，供news.html动态渲染
│   └── gallery.json        # 摄影展区内容数据，供gallery.html动态渲染
├── img/                    # 图片资源目录
│   └── gallery/            # 摄影展区图片存放目录，供photography.html/gallery.html使用
├── js/                     # 前端JS脚本
│   ├── main.js             # 全站表单校验与交互优化，服务所有页面
│   ├── admin.js            # admin.html 后台管理页交互逻辑
│   ├── profile.js          # profile.html 用户个人主页交互逻辑
│   ├── duty.js             # duty.html 值班表页面交互
│   ├── loading.js          # loading.html 加载动画页面交互
│   ├── news.js             # news.html 新闻页面交互
│   ├── photography.js      # photography.html 摄影展区页面交互
│   ├── recruitment.js      # recruitment.html 招新页面交互
│   ├── resources.js        # resources.html 资源申请页面交互
│   ├── submit.js           # submit.html 投稿页面交互
├── pages/                  # 主要功能页面
│   ├── blog.html           # 博客/公告页面
│   ├── duty.html           # 值班表页面
│   ├── forgot.html         # 找回密码页
│   ├── login.html          # 登录页
│   ├── members.html        # 成员介绍页面
│   ├── news.html           # 新闻动态页（渲染news.json）
│   ├── photography.html    # 摄影展区页（渲染gallery.json）
│   ├── privacy.html        # 隐私政策页面
│   ├── profile.html        # 用户个人主页
│   ├── recruitment.html    # 招新报名页面
│   ├── register.html       # 注册页
│   ├── resources.html      # 资源申请页面
│   ├── submit.html         # 投稿区页面

├── generate_news_json.py   # 新闻内容生成脚本（生成data/news.json）
├── generate_gallery_json.py# 摄影展区内容生成脚本（生成data/gallery.json）
└── Project_structure_description.html # 本文件，项目结构和说明

目录/文件说明

  • css/：全站和各页面样式文件，每个css文件标注其主要服务页面。
  • data/：新闻、摄影展区等内容的JSON数据，前端页面自动读取渲染。
  • img/gallery/：摄影展区图片，供photography.html和gallery.html使用。
  • js/：各页面交互脚本，每个js文件标注其主要服务页面。
  • pages/：每个页面单独标注其用途。
  • index.html：网站首页。
  • generate_news_json.py / generate_gallery_json.py：内容管理脚本，自动生成内容数据文件。
  • Project_structure_description.html：本文件，项目结构和说明。

Directory/Files Explanation

  • css/: All site and page-specific stylesheets, each file notes its main target page.
  • data/: JSON data for news, gallery, etc., auto-read by frontend pages.
  • img/gallery/: Gallery images, used by photography.html and gallery.html.
  • js/: All page interaction scripts, each file notes its main target page.
  • pages/: Each page is individually annotated for its purpose.
  • index.html: Website homepage.
  • generate_news_json.py / generate_gallery_json.py: Content management scripts, auto-generate data files.
  • Project_structure_description.html: This file, project structure and description.
